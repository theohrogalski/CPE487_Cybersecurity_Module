## cpe_487_finalProject_csecModule
*Theodore Rogalski, Ryan Manley, Mircea Florescu*
# Introduction
This project implements the SHA-256 hashing algorithm on a Nexys17100T FPGA using VHDL, with plaintexts entered using a USB keyboard and the output displayd through the VGA port on the FPGA.
# Background
The SHA-256 hashing algorithm converts a plaintext into a 256-bit hash. Details on this process are available [here](https://www.boot.dev/blog/computer-science/how-sha-2-works-step-by-step-sha-256/#step-5---create-message-schedule-w).
